# purchase-archives

Эта папка — место для хранения исходных ZIP-архивов карточек закупки.

## Что должно быть внутри одного ZIP-архива

Каждый ZIP-архив описывает **одну** карточку закупки и должен содержать:

- ровно один `.md` файл с описанием карточки (структурированный текст: номер карточки,
  статус, наименование товара, цена, количество, поставщик и т.д.);
- фотографии товара, фотографии инвойса, фото поставщика/чата (если есть);
- документы, относящиеся к этой карточке (PDF и т.д., если есть).

Файлы могут лежать как прямо в корне архива, так и во вложенной папке — импорт
обработает оба варианта.

## Важно

- Если в архиве **нет** `.md` файла или их **больше одного** — такой архив будет
  пропущен при импорте, об этом выведется предупреждение.
- Не переименовывайте файлы внутри архива вручную без необходимости — импорт
  сохраняет оригинальные имена файлов как есть.
- Сами ZIP-архивы **не публикуются** на сайте и недоступны для скачивания через
  готовый сайт — наружу попадают только распакованные изображения/документы и текст
  карточки.

## Как добавить новую карточку

1. Положите новый ZIP-архив в эту папку (`purchase-archives/`).
2. Запустите импорт (см. `china-first-purchase/README.md`).

Эту папку можно свободно пополнять новыми архивами — это её прямое назначение.
